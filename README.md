# Controle-de-Medicamentos
(Teste de Software) 3º Semestre

Software inicialmente pensado para auxiliar os familiares no controle de medicamentos.

Contendo: Adição de medicamentos, horário para tomar um medicamento, dose de medicamento, salvar quantidade de comprimidos ainda restam (controle de estoque);

Tela 1: medicamentos do dia.

Tela 2: Lista de medicamentos adicionados. (controle de estoque).

Tela 3: Cadastro de medicamento.

Tela 4: Histórico de medicamentos.

Tela 5: Aviso de quase esgotamento de medicamento (Em análise do grupo)
